# HIIT the Beat

![HIIT the Beat logo](assets/images/Logo_white_1.png)

## Overview

HIIT the Beat is a music-driven fitness product built around HIIT workouts, subscriptions, progress tracking, promo flows, onboarding, and admin management surfaces.

## Platforms

- iOS
- Android
- Web / admin surfaces are present in the export

## Store Links

- App Store: https://apps.apple.com/us/app/hiit-the-beat/id1319882523?l=ru
- Google Play: https://play.google.com/store/apps/details?id=com.breakleticsde.breakletics

## Tech Stack

- Flutter
- Dart
- FlutterFlow
- Firebase Auth
- Cloud Firestore
- Cloud Functions
- Firebase Analytics
- Firebase Crashlytics
- Firebase Performance Monitoring
- Firebase Remote Config
- Firebase Storage
- RevenueCat via `purchases_flutter`

## Notable Capabilities

- workout and promo video playback via `chewie` and `video_player`
- subscription and paywall flows
- analytics and marketing integrations including Facebook App Events
- admin panels and product configuration screens included in the export
