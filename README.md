# HIIT the Beat

![HIIT the Beat logo](assets/images/Logo_white_1.png)

FlutterFlow-exported application source for HIIT the Beat, a music-driven HIIT fitness app for different training levels.

## Stack

- Flutter
- Dart
- FlutterFlow
- Firebase
